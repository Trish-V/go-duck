🦆 The Legend of the Century

Cinematic Introduction

The Trailer of the Epic

Chapter I: The Encounter at the Crossroads

In the legendary Silicon Valley of Code, a nomadic Gopher—lightning-fast and known for his tireless concurrency—crossed paths with a Duck from the Great Persistence Bayou. The Duck held the wisdom of adaptability and the secret to navigating ever-shifting business tides. They realized that while the Gopher built fast, the Duck built to survive. Together, they forged a pact to create the Generator of Kings.

Gin Gonic Tonic

Chapter II: The Refreshment of Performance

To fuel their grand design, they sought the Legendary Bottle of Gin. This magical brew wasn't just for hydration; it transformed their web routing into a crystalline, high-performance flow. Routes became fast, middleware became transparent, and the developer experience became as refreshing as a cold tonic on a summer's day. This gave GO-DUCK its distinctive, lightweight spirit.

Chapter III: The Armor of the Divine

But speed without strength is a house made of cards. In the digital forge of the underworld, they recovered the Mark of Kratos. By stamping this sigil onto their internal services, they achieved gRPC industrial resilience. Every service became armored with strict Protocol Buffer contracts, ensuring that no matter how hard the system scaled, it would never break under the divine weight of technical debt.

Chapter IV: The GDL Genesis

Thus, the GDL (Go-Duck Language) was hatched. A single, simple tongue that could command entire legions of code. From that day forth, every developer who whispered GDL into the CLI would see their architecture evolve—bringing the Gopher's speed, the Duck's wisdom, the Gin's clarity, and the Kratos' strength into a single, unified masterpiece.

✨ Features Overview

🚀 Dual-Protocol Engine

Scaffolds side-by-side Gin Gonic REST APIs and blazing-fast Kratos gRPC services, both unified by a cleanly abstracted repository layer.

🏢 Dynamic Multi-Tenancy

True database-per-tenant isolation using a master registry. HTTP middleware securely resolves tenant context and hot-swaps PostgreSQL connection pools dynamically.

🦆 Stateful GDL Evolution

The parser maintains snapshot history to detect logical schema deltas. It atomicaly generates precise Liquibase XML migrations without destroying existing data.

🛡️ Hardened Security

OIDC/Keycloak integration for zero-trust JWT validation, CORS policies, advanced Open API permission granularity, and HMAC WebSocket payload signing.

🔍 PostgREST-like RPC

A massive dynamic search engine supporting deep JSON/JSONB arrow notation (->>) via URL queries alongside standard equality and inequality operators.

⚡ Eventing & Real-Time

Automated hook injection triggering MQTT message streams for entity changes, plus a REST-over-WS Gorilla multiplexer for high-frequency trading apps.

🧱 Resilience & Caching

Distributed tenant-aware Redis caching with automatic pattern invalidation on mutations, all wrapped in Sony/Gobreaker circuit breakers.

🔭 Full-Stack Telemetry

Seamless OpenTelemetry tracing from Gin router to the GORM database layer (otelpgx), accompanied by streaming Datadog logs and StatsD metrics.

📖 Gorgeous Docs

Zero-effort OpenAPI 3.0 documentation featuring explicit endpoint filtering, plus this exact multi-page HTML web portal built on TailwindCSS.
