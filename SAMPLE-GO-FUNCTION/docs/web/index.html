🦆 The GO-DUCK Story

Once upon a time in the silicon valley of code, a nomadic Gopher (representing the speed of Go) met a versatile Duck (the mascot of adaptability). Together, they hatched a plan to create the most evolutionary code generator in the ecosystem.

The result is GO-DUCK. It doesn't just generate boilerplate; it evolves with your business logic. We integrated a chilled bottle of Gin for the most refreshing web routing and the divine mark of Kratos to ensure your microservices are armored with enterprise-grade gRPC resilience.

💾 Installation

Install CLI Globally
$ npm install -g go-duck-cli

Pre-requisites: Node.js (v18+), Go (v1.21+), and Docker Compose for infrastructure.

🚀 Scaffold & Run

Scaffolding Command 
# 1. Create a new microservice
go-duck create -o ./my-app -c config.yaml

# 2. Enter and run
cd my-app
docker-compose up -d
go run main.go
